CICOVALE - 2023

Problema: Fazer um projeto que facilite a pesquisa das sequelas do covid-19

Resolução: Site que facilita a busca de dados de consultas, procedimentos, tratamentos, medicamentos e internações do ano de 2019 a 2022 para pesquisa de covid longa.

Tecnologias: Python, HTML, MySQL.

Contribuições: Fiz o banco de dados do site como também fui o PO (Product Owner).
